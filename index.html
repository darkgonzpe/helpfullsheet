HTML Table References

Table Tags

Tag Name What is
<table> Table Creating a table element in which the table structure will remain
<tr> Table Row The table row container.
<td> Table Data You can use it to add data to a row.
<thead> Table Heading Structure and clarify the meaning of data
<th> Table Head defines individual cells within thead rows.
<tbody> Table Body Contains the main rows of data.
<tfoot> Table Footer Footers are often used to contain sums, differences, and other data results. It is located after the closure of <tbody>

Table Attributes

Attribute Name What is
<colspan> Column span Defines how many columns a td element should span.
<rowspan> Row span Defines how many rows a td element should span.
